# CODSOFT-face_recognition
This project detects human faces in images using OpenCV's Haar Cascade Classifier.

## Objective
Develop an AI application that can detect faces in images using pre-trained computer vision models.

## Features
* Detects one or more faces in an image
* Draws bounding boxes around detected faces
* Uses a pre-trained Haar Cascade model
* Works with individual and group photos
* Beginner-friendly implementation using Python

## Technologies Used
* Python
* OpenCV
* Matplotlib
* Google Colab

## Project Workflow
1. Upload an image containing one or more faces.
2. Convert the image to grayscale.
3. Load the Haar Cascade face detection model.
4. Detect faces in the image.
5. Draw rectangles around detected faces.
6. Display the output image.

1. Open face_detection.ipynb.
2. Run all cells in order.
3. Upload an image when prompted.
4. View the detected faces in the output.
